This is a sikh prayer reading website. Once the prayer audio is played the paging happens automatically to reflect the script for the proyer being played. I was inspired to make this as i needed access to read and meditate while travelling in trains.

Technologies used

  • Yii PHP MVC Framework
  • PHP
  • MySQL
  • jQuery
  • jQueryUI
  • jAudio
  • Netbeans 7.0
