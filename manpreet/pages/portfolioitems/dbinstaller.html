DB Installer

This is an application to facilitate installation of database scripts ie. table creations, procedure creations, data insertions etc into the database. Currently this only supports MySQL database but the plan is to make this work on other popular databases as well.

Technologies used

  • Java
  • Spring Framework
  • Netbeans 7.0
  • MySQL
